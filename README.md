# Restaurant Management System

## Description
Cette application web permet de gérer des restaurants et leurs menus.
Elle offre une interface d’administration pour ajouter, modifier et supprimer des restaurants ainsi que leurs plats.

## Technologies utilisées
- Frontend : Angular
- Backend : Spring Boot
- Base de données : MongoDB

## Fonctionnalités principales
- Gestion des restaurants (CRUD)
- Gestion des menus (CRUD)
- Affichage des restaurants et de leurs plats
- Interface utilisateur interactive avec Angular

## Fonctionnalités supplémentaires
- Ajout d’images pour les restaurants 
- Soft delete des restaurants
- Support de traduction de l’interface
- Navigation dynamique entre les pages
- Gestion des modes d’éclairage
- Activation et désactivation des signaux sonores

## Lancer le projet

### Prérequis
- Node.js installé
- Java installé
- MongoDB installé

### Lancer MongoDB
Assurez-vous que MongoDB est démarré en local

### Lancer Backend (Spring Boot)
mvn spring-boot:run

### Lancer Frontend (Angular)
ng serve
